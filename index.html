Welcome

Archivo de Boards Attiny para Arduino IDE 1.6.x

Soporte para Attiny 84, 85 y 2313.

Descargar

https://github.com/Coyotus/tiny/archive/tiny.zip

Instalar

  1. Descomprimir
  2. Renombrar tiny-tiny por tiny

  3. Mover la carpeta dentro del directorio Hardware del Arduino IDE

    Ej: C:\Program Files (x86)\Arduino\hardware
